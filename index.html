Introduction

Who is a web developer?

A web developer is someone who creates a website. A developer can be a frontend (i.e the user interface of the website) developer, a backend (i.e the server-side interface of the site) developer or a full stack (i.e both frontend and backend) developer.

How do you become a web developer?

There are many documentations, sites, videos etc. that would help you and this is one of those. Though this is focused on frontend development, you can always consider other sites for other parts. Read through this documentation to guide you.

What you should already know

First of all you should already know how to at least type using a keyboard and have knowledge on how to use a system. And lots of dedication because you are going to need many of those.

If you are good with aesthetics and color mixture then you would probably make a good frontend developer but if you are good with logic then backend development would be best suited for you. Remember there is no guarantee in this assumption and of course if you feel like you got all it takes then you can always be a full stack developer.

HTML

Hypertext Markup Language (HTML) is a markup language that defines the structure of the each page in a website.

It is used to display the content of what is written into the browser. The HTML file is saved with a .html file extension.

HTML contents are written in elements with opening and closing tags. Most of these elements are nested in two sub-main elements which are the head element (for describing the content of the webpage) and the body element (for the visual part of the webpage). All of the elements are futher nested in a single parent element which is the html element. Below is a sample of a typical HTML document.

<!DOCTYPE html> <html> <head> <title>Web title</title> </head> <body> <h1>Hello World!</h1> <p>My first webpage</p> </body> </html>

Worthy of note are:

  • The opening tags - <tagname>
  • The closing tags - </tagname>

    • CSS

    Cascading Style Sheet (CSS) is a styling language used to style the HTML document. When we see designed websites, CSS was used to beautify them.

    CSS has a syntax which each styling must follow and that is - selector {property: value;}

    The 'selector' is the specified element chosen for styling, the 'property' is the chosen area of style (e.g font size, background color etc.) of the selector you want to change and the 'value' is the chosen value of the property (e.g 10px, red etc.)

    We can style the heading of a document with this example:

    h1 { font-weight: bold; background-color: grey; color: white; padding: 10px auto; }

    CSS can be added to an HTML file by:

  • Inline styling - is directly writing the styles in the HTML document using the style attribute of the selected element e.g
    • <p style="width: 150px;"> Hello World! </p>
  • Internal styling - is writing the styles in the HTML file by using the style element e.g
    • <style> button { width: 150px; height: 100px; } </style>

    Note: It is highly recommended to nest internal styles in the head element of your document

  • External styling - is writing the styles in a different file entirely and then linked to the HTML document by a self closing link element. The CSS file is saved with a .css file extension. For example the sample below is a CSS file with filename styles.css and is imported to our HTML document
    • <head> <link rel="stylesheet" href="styles.css"> </head>

    JavaScript

    JavaScript programming language is used to characterize the HTML document. That is to say it tells the elements how to behave upon user interaction.

    JS can also be internal or external and are written or linked to the HTML document using the <script> tag.

    An example of a JS code:

            
          var x = 1;
          var y = 12;
          var z = x + y;
